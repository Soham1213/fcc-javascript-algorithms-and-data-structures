# Telephone Number Validator

In this project, I created a function that returns true if the passed string looks like a U.S. mobile phone number. The passed string can be in any of the following formats:

1. 555-555-5555
2. (555)555-5555
3. (555) 555-5555
4. 555 555 5555
5. 5555555555
6. 1 555 555 5555
